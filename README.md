https://indiejapan.github.io/ip365/

日本のインディーゲームを収録・宣伝する非公式パロディサイト。
日本で作られ、IP360補助金の交付を受けていないインディーゲームを収録・宣伝します。
日本発コンテンツの海外売上を2033年までに20兆円とする目標に向け、掛け算で全体の魅力を高めます。
不採択になった作品も、そもそも申請していない作品も、ここでは等しく一作品です。
